TensorFlow.js: Using a Keras Model in the Browser

Description:

This file is based on https://github.com/tensorflow/tfjs-examples/tree/master/mobilenet.

This project was initally created in February 2020 as an assignment for COMS 4995 Applied Deep Learning, a graduate computer science course at Columbia University.

A convolutional neural network was trained on a small dataset consisting of three classes of images, all of which are landmarks at Columbia: 1) Alma Mater Sculpture, 2) Columbia Lion Sculpture, and 3) The Butler Library.

To use this tool, simply upload an image (preferably an image of one of these three Columbia landmarks), and beside the image you will then see the predicted class probabilities for your image.
